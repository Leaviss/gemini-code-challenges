## Day 1: The Story Teller (Input/Output)

**The Goal:**
Create a "Mad Libs" style program to learn how to handle standard input (stdin) and output (stdout) with string formatting.

**Requirements:**
1. Prompt the user to enter a **noun** (e.g., "gopher").
2. Prompt the user to enter a **verb** (e.g., "climb").
3. Prompt the user to enter an **adjective** (e.g., "lazy").
4. Print a complete sentence combining these words into a story.

**Example Output:**
```text
Enter a noun: gopher
Enter a verb: climb
Enter an adjective: lazy

The lazy gopher likes to climb the cloud server.
