## Day 4: The Server List (Slices & Loops)

**The Goal:**
Learn how to manage dynamic lists (slices) and iterate over them using `range`.

**Requirements:**
1. Initialize an empty slice of strings.
2. Append static values ("darkstar", "ai-core", "omni-1").
3. Prompt the user to add one more server name.
4. Iterate through the slice using `range` and print the index + server name.

**Example Output:**
```text
Enter new server name: backup-01

Server List:
[0] darkstar is online
[1] ai-core is online
[2] omni-1 is online
[3] backup-01 is online
